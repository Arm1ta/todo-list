# React Todo List

A simple and interactive **Todo List** application built with **React**, **JavaScript**, and **CSS**.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by status: `All`, `Do`, `Done`
- Responsive and interactive UI

## Technologies
- React (functional components, hooks)
- JavaScript (ES6+)
- CSS for styling
